🚀 **Student Management System** – C++ Project

This repository contains a **Student Management System** implemented in **C++** using **Object-Oriented Programming (OOP)** principles. 
It provides a clean, modular, and user-friendly console interface for managing student records efficiently.

### 🛠 Key Features
- **➕ Add student records
- **🔍 Search** students by ID or name
- **📋 Display** all student records
- **💾 File-based persistence** to save/load data
- **📁 Modular structure** with header (`.h`) and implementation (`.cpp`) files


### 🏃 How to Run
1. Open the project in a **C++ IDE** (e.g., Visual Studio, Code::Blocks).  
2. Compile all `.cpp` files together.  
3. Run the `main.exe` (or equivalent).  
4. Use the console menu to manage student records.
